## 2026-05-15 - [macOS UI Bubble Architecture & Truncation Extraction]
**Anomaly:** GTK hardcoded 7 line / 500 char logic directly into the view logic. The macOS view logic was merging into a monolithic file (`comms.rs`) lacking bubble container styling and standard truncation functionality.
**Resolution:** Abstracted string truncation to a zero-copy utility function `gneiss_pal::types::calculate_truncation`. For macOS, decoupled `chat_manager.rs` from `comms.rs` to replicate GTK's clean abstraction. Replaced standard transparent text cells with natively drawn `NSBox` components (custom type, colored by speaker role). Bound system icons (`chevron.down`, `chevron.up`) to `NSButton` actions allowing row-expanding truncation state managed via `NSIndexSet` reload.
