## 2024-05-24 - [Architectural Audit & Blueprint]
 **Structural Anomalies Found:** Previous iterations failed due to strict `objc2` v0.6.4 API boundaries, memory leaks from `block2`, missing exact generics on arrays (`NSArray<NSString>`), and void return type inference failures in `msg_send!` macros.
 **The Woz Blueprint:** Replaced `block2` with `dispatch2` for main thread routing. Implemented strict type safety, empty `unsafe impl` for missing super-protocols, and exact generic parameters for `NSArray` and `NSToolbarItemIdentifier`. Used explicit `let _: () = msg_send![...]` for void returns and stored `Retained<T>` in `RefCell` to prevent memory traps.
