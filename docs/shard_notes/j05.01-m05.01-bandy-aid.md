## 2026-04-04 - [J05.01 "Aegis" Taxonomy and Abstraction Refactor]
**Anomaly:** Cargo's `--workspace --exclude quartzite` flag was insufficient for isolating core compilation because `quartzite`'s downstream dependents forcibly dragged it into the resolution graph, causing false-positive verification failures when testing the core libraries.
**Resolution:** Explicitly whitelisted the core crates in the verification harness (`cargo check -p bandy -p gneiss_pal -p elessar -p unafs -p euclase -p lux -p resonance`). Also installed necessary native Linux development packages (like `libgtk-4-dev` and `libadwaita-1-dev`) to allow the broader graph to compile cleanly within the automated environment.
