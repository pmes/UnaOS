## 2024-10-30 - UI Constriction & Boot Sync
**Anomaly:** GTK layout lacked expansion flags, causing assertions and constraining the DAG tree view. Furthermore, Qt UI components failed to render the Matrix topology on start because they were not receiving an initial state broadcast to populate the CXX-Qt model.
**Resolution:** Replaced constrained sidebar structure by clearing "CONTEXT VECTOR" heading and unused context container logic, giving full doman to the `matrix_scroll` via `.vexpand(true)` and `.hexpand(true)`. Emitted a `TopologyMutated` boot sync event before entering the main `select!` event loop in `main.rs` to ensure that initial layout triggers state hydration in both Qt and GTK backends.
